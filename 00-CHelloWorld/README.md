# 00-CHelloWorld
Este es la resolucion del trabajo practico 0 que servia para realizar toda la instalacion de las herramientas que vamos a utilizar en los demas trabajos

## Versiones
En este programa se utilizo:
- Gcc 15.2.0
- Version de C: 202311 (estandar del 2023 / 11 de noviembre)

## El programa se compilo usando la linea:
gcc -std=c23 hello.c -o hello
## La salida se redirigio al archivo output.txt usando:
.\hello.exe > output.txt
## Makefile
Se agrego makefile con varios targets para poder compilar, ejecutar, ejecutar enviando al output y para borrar los archivos generados
